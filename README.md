# SimuSearch Studio

SimuSearch adalah platform simulasi interaktif untuk metode Information Retrieval, dibangun dengan teknologi web modern.

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (React Framework)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Shadcn/UI
- **Icons**: Lucide React

## Fitur Utama
- **Simulasi Metode IR**: Boolean, Vector Space Model (VSM), BM25, Clustering, dan Regex.
- **Visualisasi Real-time**: Highlight teks dan skor relevansi dokumen.
- **Desain Modern**: Antarmuka glassmorphism dengan tema "Modern Tech".

## Cara Menjalankan

1.  **Instal dependensi:**
    ```bash
    npm install
    ```

2.  **Jalankan server pengembangan:**
    ```bash
    npm run dev
    ```

Aplikasi akan berjalan di `http://localhost:3000` (atau port yang dikonfigurasi).

## Skrip Tersedia
- `npm run dev`: Mode pengembangan dengan Turbopack.
- `npm run build`: Build untuk produksi.
- `npm run start`: Jalankan server produksi.
- `npm run typecheck`: Validasi tipe TypeScript.
- `npm run lint`: Cek kode dengan ESLint.
